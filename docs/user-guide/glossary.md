# Glossar

Das Glossar erklärt die wichtigsten Begriffe, die in der UI und in der Dokumentation verwendet werden.

## Begriffe

- **Gebinde**: Behälter/Einheit, der Messungen zugeordnet werden.
- **Messung**: Importierte Messdaten (aktuelle Revision) inkl. ISO-11929-Tabelle.
- **Revision**: Version einer Messung bei erneutem Import derselben Mess-ID.
- **RPT**: Protokolldatei der Gammaspektrometrie.
- **NV (Nuklidvektor)**: Nuklidanteile in % für ein Jahr (Summe = 100%).
- **FMK**: Freimesskampagne (Konfiguration für Pfade/Prüfungen).
- **FGW**: Freigabewerte je Nuklid und Pfad (Einheit abhängig vom Pfad).
- **SW/KF**: Schwellenwert/Korrekturfaktor zur Anpassung von Freigabewert/Aktivität.
