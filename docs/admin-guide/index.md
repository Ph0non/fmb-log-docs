Dieser Teil richtet sich an Administratoren und Key‑User. Er erklärt die Aufgaben, die für einen stabilen Betrieb notwendig sind: Ersteinrichtung, Nutzer‑ und Rechteverwaltung, Pflege von Freigabewerten sowie Betrieb/Backup der Datenbank.
